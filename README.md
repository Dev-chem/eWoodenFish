# 介绍

电子木鱼，是一个手机应用。旨在通过敲屏幕上的木鱼来回复因种种原因（包括但不限于看地狱笑话）被扣除的功德。可以阅读腾讯网文章：https://new.qq.com/rain/a/20221025A07HO500
